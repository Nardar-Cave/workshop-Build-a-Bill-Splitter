# workshop-Build-a-Bill-Splitter

Workshop do Freecodecamp (Certificação em Python), com as práticas usando números e operações matemáticas para construir uma calculadora de divisão de contas. Essa ferramenta calculará quanto cada pessoa deve pagar após a soma dos custos da refeição e da gorjeta.


* Workshop.py contém o conteúdo que foi sendo modificado conforme as instruções (de forma gradual, conforme à ordem das solicitações).
* FinalCode.py contém o código final em python do Workshop (de forma rígida, para passar no teste).
* CodigoNovo.py contém o código atual com minhas modificações.
